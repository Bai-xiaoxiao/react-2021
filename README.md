# react-2021
2021年，React-v17.x基本语法学习


## 直接打开
script引入react库操作script引入react和reactDOM进行渲染，直接打开html


## 脚手架演示
使用create-react-app当做基础脚手架
主要引入了antd，redux，react-router-dom，用craco修改webpack配置，
cd ./react_staging

安装：
npm install/yarn install

运行：
npm start

### 关于dev环境使用node代理防止跨域
